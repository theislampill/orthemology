## Firewall

Warrant does **not** flow upward. L1's formal conformance does not evidence
L2–L6. L2/L3 deterministic validation demonstrates **representability**, not
truth. L4 is conditional on premises. L5 is creed-internal (Atharī). L6 has **no
empirical result**. Each layer cites only its own evidence status; the R7B
extensions (L2, L3) deepen the thesis without collapsing this separation.
