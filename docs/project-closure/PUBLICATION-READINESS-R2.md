# Publication readiness (R2)

**What this repository is ready to be:** a public, versioned, honestly-labeled research-stage draft state — which is what it is. **What it is NOT ready to be, and does not claim to be:** a peer-reviewed publication, an empirically validated result, a released/citable artifact with bibliographic identity, or a neutral theological conclusion.

| Readiness axis | State |
|---|---|
| Internal consistency | Machine-checked: 9 CI steps, all deterministic, all green at this revision |
| Honesty front-matter | README/STATUS/VERSION/CITING aligned; every draft PDF carries the DRAFT status page |
| Claim discipline | Claim-source matrix enforced by review; five claim levels kept separate; three-outcome rules frozen |
| Reproducibility | SHA-256 release manifest; PDF source-hash sidecars; registry-driven validators; frozen packet hashes |
| Privacy | Review D clean; historical headers quarantined to provenance docs; no binaries but generated PDFs |
| Legal | **Blocked on owner:** license, identity (by design — inventing them would be fabrication) |
| Empirical | **Blocked on owner:** nothing runnable without spend/participants; nothing is claimed run |
