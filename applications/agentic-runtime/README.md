# Agentic runtime application outlines

**Status: outline-only. Execution: not implemented. Runtime owner: external/downstream.**

This directory places possible runtime consumers of the repository's waking and somnic contracts without turning them into skills or a local runtime. The inventory separates capture, claimant gating, residual recording, recurrence assessment, intervention disposition, proposal generation, actuation, orchestration, transclusion, and council comparison. The writeback profile treats guarded proposal/review/dry-run/apply/revert machinery as a downstream actuator. The collective profile separates independent convergence, federated transclusion with local re-orthing, and intentional council coordination.

No file here is an executable skill, scheduler, network, transport, shared-memory service, writeback engine, or automatic adoption mechanism.
