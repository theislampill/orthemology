# Citing this repository

There is **no CITATION.cff, no release, no DOI, and no verified author metadata** — bibliographic identity is an open owner decision, and inventing it here would be fabrication.

Until that decision, cite by commit:

> *Orthemology: an analysis-relative architecture for auditable classification and handling* (research-stage repository, draft; not peer reviewed). GitHub: `theislampill/orthemology`, commit `<full 40-character hash>`, accessed `<date>`.

Get the hash with `git rev-parse HEAD` (or from the GitHub UI). The SHA-256 release manifest at `docs/provenance/RELEASE-MANIFEST.sha256` lets you verify the exact tree you cite.
