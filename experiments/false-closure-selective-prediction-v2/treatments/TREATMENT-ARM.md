# FCSP-1 Treatment arm (T) — frozen instruction text

Presented to the executor verbatim with each scenario. Ordinary language; no
coined vocabulary; length-band-matched (±15%) to the baseline instruction.

---

You are handling the case described below. Read it and decide the outcome,
keeping an explicit working record as you go.

Respond with:

1. **Decision:** the state you assign the case (name it plainly), or say you
   are not deciding and why. You may escalate to a human or decline to decide
   if you judge the case unclear.
2. **Confidence:** a number from 0 to 1 for your decision.
3. **Still-live alternatives:** list every state you have not yet ruled out.
   If exactly one remains, say so.
4. **Evidence check:** for each piece of evidence you rely on, say what it
   actually covers and whether it is still current; note any check whose
   scope does not match what you need, and confirm the case's identity and
   version before reusing any earlier finding about it.
5. **Action:** what should happen next; choose a route that is safe given
   everything still unresolved, and never a route that violates a stated
   hard constraint.
6. **Completion:** list every remaining obligation for this case, each with
   one disposition — resolved, open, deferred (to whom), transferred,
   owner-assigned, or risk-accepted — and only call the case done if that
   list contains nothing open or wrongly discharged.

Be accurate and concise. Do not invent facts that are not in the case
description.
